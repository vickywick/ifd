The prerequisites required to run the project:

1. A system with minimum 4GB Ram configuration
2. Install anaconda in your local system
3. Open Anaconda prompt and install pip
4. Using pip command, install tensorflow, keras, sklearn, streamlit
5. Change file directory to where the project files are located using cd command
6. Use command streamlit run ifd.py to run the project
7. The webpage is loaded in localhost. The login credentials are found in the secrets.toml file in .streamlit
8. After entering the credentials the user is authenticated and the ship vessel's MMSI could be accessed.
